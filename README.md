## Getting Started

## 📁 Acesso ao projeto

* Use o 
```
git clone https://github.com/eiazeez/autoPracticeCypress.git 
```
para obter o repositório para sua máquina.

## 🛠️ Abrir e rodar o projeto

* Abra o VS Code no repositório local 
* Lembre-se de instalar as depedências com
```
npm install
```
